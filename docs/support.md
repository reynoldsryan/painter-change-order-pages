# Painter Change Order Support

Painter Change Order helps residential painters package extra work into a cleaner client-ready request before unpaid labor starts.

## What The App Does

- Lets you start from a painter-specific template.
- Supports `1-4` job-site photos plus a short scope note.
- Adds one price adjustment and one time-impact field.
- Generates a clean request card and PDF ready to share by text or email.
- Saves sent requests locally for review and repeat use.

## Important Limitation

Painter Change Order is a lightweight communication tool. It does not include signatures, approval tracking, invoicing, CRM workflows, sync, or a backend service.
